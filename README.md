# kindでk8sを試すサンプル  
kindでローカルにk8sの開発環境を構築し、  
Go言語のAPIをデプロイするサンプルです。  
  
## 機能一覧  
・テキスト表示１（GET /）  
・テキスト表示２（GET /k8s-go）
  
## 使用技術  
Go "1.24"  
Echo "v4.13.3"  
Docker  
docker-compose  
kubectl  
kind  
k8s
